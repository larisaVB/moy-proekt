Добро пожаловать на курс «Разработка сайтов»

Курс для тех, кто хочет создавать сайты и стать профессиональными программитстом.

На курсе вы разберете работу с популярными технологиями для Frontend-разработчика: HTML, CSS, JS, ReactJS. 34rontend-программисты могут работать в разных условиях: от удаленного формата до офисного.

JavaScript — язык программирования, который применяют в веб-разработке. С его помощью сайты делают интерактивными: добавляют всплывающие окна, анимацию, кнопки лайков и формы для отправки информации. Его ещё называют главным языком фронтенда — «лицевой» стороны сайта, с которой взаимодействуют пользователи.
